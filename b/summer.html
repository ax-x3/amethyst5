Summer Coding @ DS

Free coding classes hosted by DS Robotics Captains

Sign up for Summer 2025!

> Learn Useful Skills...

Join us this summer to learn how to make games, write code, or build websites! Learning to code is fun, and it can help you build a career path into computer science. This is also a great way to get yourself noticed by the DS Robotics Team, should that interest you.

We're looking mainly for 4th to 9th grade students, but we're willing to accommodate anybody who wants to learn.

If you're even a little bit interested, go ahead reserve a spot. You can always change your mind later.

> Hosted With Experience...

This free class is run by Amethyst and Dan, two captains of the DSHS Robotics Team. Amethyst is a rising highschool senior, and Dan just got accepted to the Rochester Institute of Technology.

This is the first summer for Coding @ DS, but we've run classes before. For the last two years, we worked through the DSHS Robotics Team to organize a robotics club in the DS Middle School. We also ran a three-week program to help Girl Scouts earn their programming and robotics badges.
